Inline Elements

Inline Elements do not have a line break.

Inline Element 1: span is a inline element. Inline Element 2: a tag is also a inline element. a tag Inline Element 3: emphasize or em tag is also a inline element. Inline Element 4: image or img tag is also a inline element. Inline Element 5: b tag is also inline element.

Block Elements

Block Elements have a line break by default

Block Element 1: All heading tags are block elements

    Block Element 2: ul tag used for unordered list is also a block element
    Block Element 3: ol tag used for ordered list is also a block element

Block Element 4: p tag is also a block element

Block Element 5: div is also a block element in html
